🦙 Ollama Model Manager

Interactive Python helper for finding, downloading, and updating Ollama models.

Python 3.10+ Standard Library Only Open Source
⭐ View on GitHub

Quick Start

Requires Python 3.10+ and Ollama installed.

git clone https://github.com/jlaiii/ollama-model-manager.git
cd ollama-model-manager
python ollama_model_manager.py

Features

🔍 Auto-Discover

Detects installed models and skips duplicates

🌐 Live Catalog

Refreshes from ollama.com/library with offline fallback

📦 Presets

Recommended, small, coding, reasoning, embedding, vision

🔄 Update

Re-pull installed models to check for updates

🧪 Dry Run

Preview what would download before committing

⚡ Zero Deps

Uses only the Python standard library

Usage Examples

Interactive picker

python ollama_model_manager.py

Preview recommended starter set

python ollama_model_manager.py --preset recommended --dry-run

Download recommended models

python ollama_model_manager.py --preset recommended

Download specific models

python ollama_model_manager.py --models llama3.2:3b qwen3:8b

Download entire catalog

python ollama_model_manager.py --all

Update installed models

python ollama_model_manager.py --update-installed

Available Presets

  • recommended — practical starter set
  • small — lightweight models
  • coding — coding-focused models
  • reasoning — reasoning-focused models
  • embedding — embedding models
  • vision — vision/multimodal models

Useful Options

--all              Queue every model in the catalog
--preset NAME      Use a preset (recommended, small, coding, etc.)
--models ...       Queue explicit model names
--update-installed Re-pull installed models to update them
--dry-run          Show what would happen without downloading
--offline          Skip online catalog refresh
--online-limit N   Limit library families to inspect
--force            Pull even if already installed
--yes              Skip confirmation prompts
